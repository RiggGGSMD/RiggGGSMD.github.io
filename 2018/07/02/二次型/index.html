前言

本章内容较少，掌握主要的计算方法即可，主要的内容分为二次型标准化与正定性的相关问题。

二次型标准化

了解一些常见的概念性质

  1. 标准二次型：只含有平方项不含交叉项的二次型。
  2. 规范二次型：系数为1或者-1的标准型。
  3. 惯性指数：标准型中系数不为0的个数以及正系数与负系数的个数是唯一的，分别为二次型的秩、正惯性指数与负惯性指数。
  4. 矩阵合同：设 A ， B 为 n 阶实对称矩阵，若存在可逆矩阵P，使得 $P^TAP = B$，称矩阵 A 与 B 合同。

二次型标准化的方法

  1. 配方法。
  2. 正交变换法。

正定二次型的判别与证明

  1. 顺序主子式法(若都大于零，则为正定矩阵)。
  2. 求特征值(若全为正，即为正定矩阵)。
  3. 定义法。
  4. 若存在可逆矩阵 P ，使得 $A = P^TP$ ，则 A 是正定矩阵。
