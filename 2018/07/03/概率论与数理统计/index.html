概率论与数理统计

前言

由于本部分所占比重不多，内容也比较简单，所以作一个总的概括，把主要的或者易忘的知识点公式罗列一下。

随机事件及其概率

  • A + A = A
  • AA = A
  • $A \Omega = A$
  • (补概率公式)$P(\overline{A}) = 1 - P(A)$
  • (减法公式)$P(A - B) = P(A \overline{B}) = P(A) - P(AB)$
  • (加法公式1)P(A + B) = P(A) + P(B) - P(AB)
  • (加法公式2)P(A + B + C) = P(A) + P(B) + P(C) - P(AB) - P(AC) - P(BC) + P(ABC)
  • (条件概率1)$P(B | A) = \frac{P(AB}{P(A)}$
  • (条件概率2)$P(A + B | C) = P(A | C) + P(B | C) - P(AB | C)$
  • (条件概率3)$P(\overline{B} | A) = 1 - P(B | A)$
  • (乘法公式)$P(AB = P(A) P(B | A)(P(A) > 0) = P(B) P(A | B)(P(B) > 0)$
  • (两随机事件独立)$P(AB) = P(A) P(B)$
  • (全概率公式)
  • (贝叶斯公式)

一维随机变量及其分布

分布函数

  • $0 \leq F(x) \leq 1, F(- \infty) = 0, F(+ \infty) = 1$
  • 单调不减
  • 处处右连续

    分布函数的充要条件是以上性质全部成立

0-1分布、二项分布、泊松分布、几何分布、超几何分布(离散型)

  • $X ~ B(1, p), P{X = k} = p^k (1 - p)^{1 - k},k = 0,1$
  • $X ~ B(n, p), P{X = k} = C_n^k p^k (1 - p)^{n - k},k = 0,1,\cdots,n$
  • $X ~ P(\lambda), P{X = k} = \frac{\lambda^k}{k!} e^{-\lambda},k = 0,1,2,\cdots, \lambda > 0$
  • $P{X = k} = p (1 - p)^{k - 1},k = 1,2,\cdots, 0 < p < 1$
  • $P{X = k} = \frac{C_{N - M}^{n-k} C_M^k}{C_N^n}$

密度函数

  • $f(x) \geq 0$
  • $\int_{-\infty}^{+ \infty} f(x) {\rm d}x = 1$

    概率密度函数的充要条件是以上性质成立

均匀分布、正态分布、指数分布(连续型)

二维随机变量及其分布

联合分布函数

  • $0 \leq F(x, y) \leq 1$
  • F(x, y)关于 x 和 y 都是单调不减的函数
  • F(x, y)关于 x 和 y 都右连续
  • $F(- \infty, - \infty) = F(- \infty, + \infty) = F(+ \infty, - \infty) = 0, F(+ \infty, + \infty) = 1$

    具备以上四个性质的二元函数也为二元随机变量的联合分布函数

数字特征

  • 0-1分布, $X ~ B(1, p), E(x) = p, D(x) = p(1 - p)$
  • 二项分布, $X ~ B(n, p), E(x) = np, D(x) = np(1 - p)$
  • 泊松分布, $X ~ P(\lambda), E(x) = D(x) = \lambda$
  • 参数为p的几何分布, $E(x) = \frac{1}{p}, D(x) = \frac{1 - p}{p^2}$
  • 均匀分布, $X ~ U(a, b), E(x) = \frac{a + b}{2}, D(x) = \frac{(b - a)^2}{12}$
  • 指数分布, $X ~ E(\lambda), E(x) = \frac{1}{\lambda}, D(x) = \frac{1}{\lambda^2}$
  • 正态分布, $X ~ N(\mu, \sigma^2), E(x) = \mu, D(x) = \sigma^2$
  • 协方差1, $Cov(X, Y) = E(XY) - E(X) E(Y)$
  • 协方差2, $Cov(X, X) = D(X), Cov(X, C) = 0$
  • 方差1, $D(X \pm Y) = D(X) + D(Y) \pm 2Cov(X, Y)$
  • 方差2, $(X + Y + Z) = D(X) + D(Y) + D(Z) + 2Cov(X, Y) + 2Cov(X, Z) + 2Cov(Y, Z)$
  • 相关系数1, $\rho_{XY} = \frac{Cov(X, Y)}{(\sqrt{D(X)} \sqrt{D(Y)})}$
  • 相关系数2, $\rho_{XY} \leq 1$
  • 相关系数3, $X 与 Y 不相关 \Leftrightarrow Cov(X, Y) = 0 \Leftrightarrow D(X \pm Y) = D(X) + D(Y) \Leftrightarrow E(XY) = E(X) E(Y)$
  • 相关系数4, $|\rho_{XY}| = 1 \Leftrightarrow P{Y = aX + b} = 1, a \ne 0, 则$
  • 相关系数5, X, Y相互独立一定不相关，反之不对
