前言

矩阵是线代中非常重要的一部分，它基本上贯穿了线代考点的所有内容，掌握好矩阵的概念，利用矩阵的性质理论去解决问题。

以下是关于矩阵的运算，主要包括了矩阵的加法，乘法，转置，行列式，共轭。

  • $A + B = B + A$
  • $( A + B ) + C = A + ( B + C )$

  • $( \lambda A )^T = \lambda A^T$
  • $( AB )^T = B^T A^T$

方阵的行列式

定义：由 n 阶方阵 A 的元素所构成的行列式(各元素的位置不变)，称为方阵 A 的行列式，应当注意的是方阵的行列式是一个数表，而行列式是一个确定的数。
应当满足下述运算规律

  • $| A^T | = | A |$
  • $| \lambda A| = \lambda^n | A |$
  • $| AB | = | A || B |$

  • $\overline{ A + B } = \overline{ A } + \overline{ B }$
  • $\overline{ \lambda A } = \overline{ \lambda }\ \overline{ A }$
  • $\overline{ AB } = \overline{ A }\ \overline{ B }$

逆矩阵及伴随矩阵

这一块在矩阵的知识中十分重要，我个人觉得需要记住的是逆矩阵的相关定理以及一些运算公式，还有伴随矩阵的定义等(若有其他之后会补充)，以方便计算与证明，下面就简单的罗列一下主要的定理以及常用的公式。

  • 定理1 若矩阵 A 可逆，则$| A | \neq 0$
  • 定理2 若$| A | \neq 0$，则矩阵A可逆
  • 推论 若 AB = E，则$B = A^{-1}$
  • $( A^{-1} )^{-1} = A$
  • $( \lambda A )^{-1} = \frac{ 1 }{ \lambda } A^{-1}$
  • $( AB )^{-1} = B^{-1} A^{-1}$

总结

矩阵这部分的基础知识大概就是这样，进行了简单的概念定义定理的梳理，对需要用到的定理性质做了整理，剩下的就是对题型的总结，需要经过大量的练习掌握各个题型的解题方法，需要精通其中的解题套路，下面再将矩阵所涉及到的性质定理进行汇总，如下(★为常用)。

  • 矩阵
    • $A + B = B + A$
    • $( A + B ) + C = A + ( B + C )$
    • $( \lambda \mu ) A = \lambda ( \mu A )$
    • $( \lambda + \mu ) A = \lambda A + \mu A$
    • $\lambda ( A + B ) = \lambda A + \lambda B$
    • ( AB ) C = A ( BC )
    • $\lambda ( AB ) = A ( \lambda B) = ( \lambda A ) B \quad(其中\lambda 为数)$
    • A ( B + C ) = AB +AC
  • 转置
    • $( A^T )^T = A$
    • $( A^T )^m = ( A^m )^T$
    • $( A \pm B )^T = A^T \pm B^T$
    • $( \lambda A )^T = \lambda A^T$
    • $( AB )^T = B^T A^T$ ★
  • 行列式
    • $| kA | = k^n | A |$ ★
  • 逆矩阵
    • $A^{-1} = \frac{ 1 }{ | A | } A^*$ ★
    • $| A^{-1} | = \frac{ 1 }{ | A | }$ ★
    • $( A^{-1} )^{-1} = A$
    • $( \lambda A )^{-1} = \frac{ 1 }{ \lambda } A^{-1}$ ★
    • $( AB )^{-1} = B^{-1} A^{-1}$ ★
    • $( A^T )^{-1} = ( A^{-1} )^T$
    • $( A^n )^{-1} = ( A^{-1} )^n$
  • 伴随矩阵
    • $A^* = |A| A^{-1}$ ★
    • $| A^* | = | A |^{n - 1}$ ★
    • $( A^T )^{\ast} = ( A^{\ast} )^T$
    • $( kA )^{\ast} = k^{n - 1} A^{\ast}$ ★
    • $( AB )^{\ast} = B^{\ast} A^{\ast}$ ★
