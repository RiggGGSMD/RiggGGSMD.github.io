线性方程组
2018-06-27|学习考研|
字数总计: 532|阅读时长: 2 分钟

前言

前段时间搬家加上学校的相关事宜导致没有时间进行总结，现在利用上午的时间对这部分进行以下小结。这部分主要的内容是线性方程组的求解问题，矩阵的知识在这章也十分重要，除了一些性质的使用，更多的是矩阵秩的应用，我的理解主要分为三个部分，解的定理，解的结构，还有一些其他的扩展之类的，主要还是理解加上练习，计算的内容比较多，也有部分证明题。

解的基本定理

解的结构

  • 设 $X_1, X_2, \cdots, X_s$ 为齐次线性方程组 Ax = 0 的一组解，则 $k_1X_1 + k_2X_2 + \cdots + k_sX_s$ 也为Ax = 0 的解，其中的k为任意常数。
  • 设 $\eta_0$ 为非齐次线性方程组 Ax = b 的一个解，$X_1, X_2, \cdots , X_s$ 为齐次线性方程组 AX = 0 的一组解，则 $k_1X_1 + k_2X_2 + \cdots + k_sX_s + \eta_0$ 为 Ax = b 的解。
  • 设 $\eta_1$，$\eta_2$ 为 非齐次线性方程组 Ax = b 的两个解，则 $\eta_2 - \eta_1$ 为齐次线性方程组 Ax = 0 的一个解。
  • 设 $X_1, X_2, \cdots , X_s$ 为非齐次线性方程组 Ax = b 的一组解，则 $k_1X_1 + k_2X_2 + \cdots + k_sX_s$ 为 Ax = b 的解的充分必要条件为 $k_1 + k_2 + \cdots + k_s = 1$。
  • 设 $\eta_1, \eta_2, \cdots ,\eta_s$ 为非齐次线性方程组 Ax = b 的一组解，则 $k_1 \eta_1 + k_2 \eta_2 + \cdots + k_s \eta_s$ 为齐次线性方程组 Ax = 0 的解的充分必要条件是 $k_1 + k_2 + \cdots + k_s = 0$。

主要的题型

在使用这些性质定理去求解时主要会遇到的题型如下

  • 求方程组的基础解系以及通解
  • 通过解的基本定理的应用去判断方程组解的情况以及求得基础解系或通解
  • 同解方程组问题
  • 证明题

总结

在遇到这部分题时要先判断方程组的类型然后再根据相应的定理性质进行解题，求解一般不难，难在理解以及对定理性质的熟练应用上，需要多加练习熟练掌握。
